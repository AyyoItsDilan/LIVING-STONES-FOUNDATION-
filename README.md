# LIVING-STONES-FOUNDATION-
Living Stones Project
● Conducted land-use change analysis for projects GREEN VERDE and CultiveConnect, using R for statistical modeling 
and visualization to build an interactive Tableau dashboard showcasing trends in agricultural and forest land over 60+ 
years. 
● Presented insights coordinators of the project linking land-use trends with macroeconomic conditions, recommending 
agricultural expansion opportunities (Canada, Uruguay, Panama) and forest restoration areas (Nicaragua, Paraguay, 
Jamaica) with backed up identified sustainable agricultural practices such as protective covers.  
● Validated and corrected misleading data, ensuring findings improved decision-making for carbon-reduction and 
sustainable development efforts. 
● Extracted and cleaned raw datasets from FAOSTAT using Python, ensuring accuracy and usability for sustainability 
research.
